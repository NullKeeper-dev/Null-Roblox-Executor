## Description
Null Roblox Executor is a sophisticated WPF-based application designed to provide a robust environment for executing Lua scripts. It features a modern user interface, integrated script editor with syntax highlighting, and seamless process integration for advanced automation tasks.

## Tech Stack
*   **Framework:** .NET 8.0 / WPF (Windows Presentation Foundation)
*   **Language:** C#
*   **Editor:** AvalonEdit for high-performance code editing
*   **Scripting:** Lua support with custom highlighting and completion
*   **API:** VelocityAPI for core execution logic

## Key Features
*   **Advanced Script Editor:** Full Lua syntax highlighting and intelligent code completion.
*   **Real-time Status Monitoring:** Automatic detection and attachment to target processes.
*   **Autoexec Support:** Automatically run scripts upon successful attachment.
*   **Script Management:** Easily open, edit, and save Lua scripts with a built-in file manager.
*   **Modern UI:** Clean, dark-themed interface with intuitive controls.

## Installation
1.  Ensure you have the [.NET 8.0 Runtime](https://dotnet.microsoft.com/download/dotnet/8.0) installed.
2.  Download the latest release from the [Releases](https://github.com/NullKeeper-dev/Null-Roblox-Executor/releases) page.
3.  Extract the contents to a folder of your choice.
4.  Run `Null Launcher.exe`.

## Usage
1.  Launch `Null Launcher.exe`.
2.  Open up Roblox (Fishtrap suggested).
3.  Click **Attach** to establish a connection.
4.  Write or load a Lua script in the editor.
5.  Click **Execute** to run your script.

```lua
-- Example script
print("Hello from NullKeeper")
warn("Automation started...")
```
